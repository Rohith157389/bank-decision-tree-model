 Bank Marketing Dataset - Decision Tree Classifier

This project builds a Decision Tree machine learning model to predict whether a customer will purchase a product or service based on demographic and behavioral data from the Bank Marketing dataset.

---

Project Objective
The goal of this project is to:
- Preprocess and clean the dataset
- Encode categorical variables
- Train a Decision Tree classifier
- Evaluate the model performance
- Interpret results and visualize the model
